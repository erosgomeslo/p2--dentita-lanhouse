##  Funcionalidades

- **Design**: Interface moderna com efeitos de Glassmorphism e animações suaves.
- **Mini-game**: Mini-game interativo integrado diretamente na página para entretenimento.
- **Responsividade**: Totalmente adaptado para dispositivos móveis e desktops.

##  Tecnologias Utilizadas

- **HTML5** & **CSS3**
- **Tailwind CSS** (via CDN)
- **JavaScript** (Vanilla)
- **Google Fonts** (Inter)
- **Lucide Icons** (SVG)

##  O mini-Jogo

Ele possui:
- Movimentação por setas do teclado.
- bots com movimentação aleatória.
- Detecção de colisão (Game Over automático).
- Gráficos em estética Neon.
